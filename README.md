# Ticket Management System 🎟️

Este proyecto es una aplicación para la gestión de tickets, diseñada para facilitar el registro, seguimiento y resolución de incidencias de manera eficiente.

## 🚀 Características
- CRUD completo de tickets (crear, leer, actualizar, eliminar).
- Validación de usuarios con login seguro.
- Interfaz intuitiva con **WinForms**.
- Conexión a base de datos mediante **ADO.NET** y **SQL Server**.
- Arquitectura en capas para mayor mantenibilidad.

## 🛠️ Tecnologías utilizadas
- **C#**
- **SQL Server**
- **ADO.NET**
- **WinForms**
- Patrón **MVP** y arquitectura en capas
